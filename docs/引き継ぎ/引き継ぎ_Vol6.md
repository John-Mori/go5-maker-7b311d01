## 1. Vol.5 でやったこと（v239 → v270 の要約）

Vol.5 は**ユーザー起点のUI微修正の大量イテレーション**（候補タブ／投稿履歴カード／各モーダル）と、**クリック計測の自己除外**、**1件のバグ根治**が中心。GASは無改修。

### A. 投稿履歴カード（v239〜v242）
- **フッター整理**：🗑をカードの真の右端（画像列の真下）へ／編集をBskyと同じ段へ。
- **定型タグ非表示（v241）**：`Go5Util.POST_TAGS` ＋ `stripPostTags`（**トークン単位**で除去）を新設。`#anime` が `#animeedit` を割って `edit` が残る旧バグを根治。動画に焼く題名・保存ファイル名・カード題名から6タグを除去（YouTube題名には残す）。
- **🛠️編集モーダルに投稿画像添付（v242）**：履歴アイテム単位（videoId/itemKey）の画像ストアを IndexedDB `post:` 名前空間に追加（`Go5Cand.postImgs/postImgSave`）。複数枚・1枚目をカード表示・タップでズーム。ズームのページ数カウンタを白字常時表示に。

### B. 候補カード 大改修（v243〜v269・最多イテレーション）
- 動画生成用画像を作品サムネの真下（左列）へ／点線区切り廃止。
- **投稿日✔チップ**：投稿済み作品に「YYYY/M/D ✔」をチャンネルテーマ色（acc1=紫/acc2=葡萄）で表示。太字。
- **投稿済みの太枠**：カード大枠をチャンネル色で囲む。両channel投稿は**月詠み(外)＋宵桜(内)の二重**（border＋box-shadowリング）。
- 作品リンク(作品↗)色を投稿履歴と同期（ピンク `#ff79b0`）。X↗リンクがBlueskyの時は **`B↗`**（旧「🦋 Bsky↗」）。2つ目URL＝`twitterUrl2` → カードに `X2↗`/`B2↗`。
- **コメント/メモの表示**：全幅・**必ず1行**（`white-space:nowrap`＋描画後 `fitOneLineTexts_` で幅に収まらない時だけフォント可変縮小。下限7px。折り返さず極力省略しない）。メモはコメントの上・水色。
- **無コメント時（`cand-nocomment`）**：非表示/🗑を**作品リンク行(cand-info定位置)の右端に統合**＝別行を作らず余白最小化。収めるため無コメント時の作品リンク行をコンパクト化（gap/padding/font）。投稿編集ボタンは「投稿編集」（🖼/✓削除）。
- 非表示ボタンの🙈削除＋押下時に「非表示にしますか？」確認ダイアログ（両描画経路）。

### C. 候補追加モーダル（v254 / v256 / v263）
- タイトルの白枠除去・白文字・余白削減。右上×を1.5倍。
- メモ用テキストボックス追加（入力があれば候補のメモへ保存）。ボタンを「候補に追加 / 続行」「候補に追加 / 閉じる」に。ボタン幅は内容追従（`width:max-content`）。閉じるはサークル追加と並列・右寄せ。
- URL欄の枠色＝作品/サークル=ピンク・X/Bluesky=水色、貼付ボタンも同色＋白字。サークル説明を2行化。**括弧は全て半角 `()`**。

### D. 投稿編集モーダル / 画像ズーム（v247 / v250 / v258 / v260 / v262）
- 作品名2行省略・白枠除去。「メモ・URL追加」小モーダル（メモ＋2つ目URL）。
- 追加画像を**1枚目に配置するのを標準**（貼り付け=先頭へ）。ズームに「この画像を1ページ目にする」（2ページ目以降）と「**画像を貼り付けて新規追加**」（クリップボード画像を先頭へ・**投稿編集と同じrefimgに同期保存**）ボタン。
- ズーム：×を1.5倍（丸背景撤去）・ページ数2倍・貼付ボタンは左寄せ固定でピンク。

### E. 動画作成タブ（v251〜v253）
- 狙い/コメント型の絵文字（🎯💬）削除・ラベル＋ドロップダウンを1行化。
- **④コメント「2行モード」**：横にチェックボックス。ONで textarea 2行化。**ユーザーの改行(\n)位置で2行目**（グリーディ折返し廃止）。両行中央・帯は1枚に統一・帯位置分離なし。**保存ファイル名は改行を無視**して連続文字列。

### F. その他
- **タブ復元（v250）**：`go5_active_tab` を保存し `DOMContentLoaded` で復元＝リロード/再訪で動画作成に強制的に戻らない。
- **バグ根治（v254）＝「月詠み✔なのに投稿日が出ない」**：原因＝シート復元で `postedAt` 空だと `ts:0` で復元され日付非表示。`idgen.js` に **`tsOfId()`** 追加（背骨ID `acc-YYYYMMDD-HHMM-rand` から作成日時≒投稿日を復元）。`postedDatesHtml_` は `ts→postedAt→videoId(tsOfId)` の順に頑健化。復元時も videoId から ts をバックフィル（再発防止）。
- **クリック計測の自己除外（v267 / v270・★worker変更あり）**：`link-worker` の `/<code>` で Cookie `go5nc=1` があるか `?nc=1` があれば **bumpClick をスキップ**（自分のクリックを数えない）。`?nc=1` で除外Cookie付与（2年）。**詳細設定タブに「🚫 自分のクリックを計測から除外(この端末)」ボタン**（`WORKER_URL/?nc=1` を開いてCookieセット）。iOSはSafariとアプリ内ブラウザでCookie共有＝Bluesky経由の自分クリックも除外。**worker はデプロイ済**。端末/ブラウザごとに1回ボタンを押す運用。

## 3. 反映・運用のルール（Vol.6でも厳守）

- **フロント変更後は `index.html` の `?v=N` を必ず上げる**（Editツールの `replace_all` で `?v=270`→次番号。分類器がPowerShell/Bashを断続ブロックする時間帯があるため）。次は **`?v=271`**。
- **★UI表示文言の括弧は必ず半角 `()`。全角 `（）` 禁止**（ユーザー明示ルール・memory `go5-halfwidth-parens-rule` 参照）。JSコメント内の全角は表示されないので許容。
- **タブ構成が変わった**：`tabVerify` の表示名は **「📋 投稿履歴」**（旧「検証」）。設定系のUIは **⚙詳細設定(`pageSettings`)** に置く。
- **GAS変更時**は `GAS_VERSION` を上げてから `node scripts/deploy_gas.mjs`／`?ping=1`照合。**本番デプロイは毎回ユーザーの明示承認が必要**（auto mode分類器がブロック＝「任せる/進めて」では通らない）。`.gs`は`node --check`不可→`.js`にコピーして構文確認。
- **link-worker 変更時**は `cd link-worker && npx wrangler deploy`（Cloudflare。**Claudeからは実行不可＝ユーザーが実行**）。GET確認は WebFetch か curl（`/?nc=1` 等）。
- **GASへのPOSTはcurlだとリダイレクトで411**→ブラウザ`fetch`（preview_eval）。GETのhistory/diagnose/pingはcurlでOK。
- **デプロイ確認は必ずlive照合**：`curl .../index.html?cb=<ts> | grep '?v='`。Pagesが `building`/失敗する事あり→`gh run rerun <id> --failed` か `gh api -X POST repos/John-Mori/go5-maker-7b311d01/pages/builds`。
- **実装完了後は `git commit && git push` まで**。コミット末尾に `Co-Authored-By: Claude Opus 4.8 <noreply@anthropic.com>`（fable使用時は Fable 5）。
- **プレビュー検証の作法**：ビューポートが0×0に潰れることがある→`preview_resize` mobile／`preview_eval`で寸法実測。スクショはbg videoでタイムアウトしがち。**CSSはブラウザキャッシュに注意**＝`style.css?v=N` は同一Nだと旧版が残る→検証時は `fetch('style.css?v=N&cb=…')` で新CSSを iframe に注入するか、`?v` を上げてから測る（Vol.5で何度も踏んだ）。候補カード等の実データが要る検証は `localStorage.cand_items` ＋ `Go5Idb.set('ref:<cid>',…)` を仕込み→リロードでハイドレート→`Go5Cand.render()`。
- **配色ルール**：Claude風の紫UI禁止。アカウントテーマ（acc1=月詠み/宵藍系 / acc2=宵桜/葡萄梅系）。ピンク=`#ff6b8a`/水色=`#5ec6e8` を Vol.5 で多用。
- **ボタン幅の恒久ルール**：`width:auto`/`max-content` でテキスト幅に追従。全幅は主要CTA/リスト行のみ。インライン`width:100%`をボタンに付けない。
- **不変条件**：比率座標系（W=1080/H=1920）・**前景の見た目/演出**（コンサル手法）・drive-worker非破壊・link-worker `u:<code>`不変・GASの`SHARED_SECRET`未設定・秘密はSecrets/Propertiesのみ。**videoIdは作成時に`acc1-`/`acc2-`接頭辞＝所属の権威シグナル**（T1/T5/T10 と tsOfId の根幹）。
