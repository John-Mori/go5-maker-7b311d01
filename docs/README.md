## 現在の短縮URL構成（要約）
- `最終URL → r2ワーカー短縮(計測用) → da.gd短縮(表示用16字) → 概要欄`
- 計測は常に r2 側。表示/コピー/シート「共有URL」列は da.gd。
- コードの要点：bluesky.js の `USE_DAGD_CHAIN`（ロールバック）/ `SHARE_SHORTENERS`（差し替え）/ `WORKER_URL`。
