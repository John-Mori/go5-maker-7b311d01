## 0. 結論

**BANではない。da.gd というサービス自体が落ちている。**
ただし「復旧を待てば直る」とも言い切れない(§3)。**53本の収益導線が死んだままなので、
復旧を待つ運用はもう続けない方がいい**(§4)。

★注意: 「BANではない」は**da.gdについての結論**であって、
**YouTube/FANZA側で何も起きていないという意味ではない**(§3.3)。

## 2. なぜ「BANではない」と言えるか(4つの根拠)

1. **`/help` と `/ip` も同じ502**。これは da.gd 自身の汎用ページで、Chamiのアカウントとも
   短縮コードとも一切関係が無い。**BANなら他人のページまで道連れにはならない**。
2. **別のIP・別の国からも同じ502**。IP単位のブロックなら、こちらの回線からだけ失敗するはず。
3. **Cloudflareのエラー画面が `You=✓ / Cloudflare=✓ / Host da.gd=✗` を出している**
   (Chamiのスクショ)。=**CDNは生きていて、その後ろの本体サーバーが死んでいる**。
   ★BANは本体サーバーが**生きていて**「お前は拒否」と返す行為(403/404/410)。
   本体が死んでいる状態は、BANを出す主体そのものが不在ということ。
4. **`Retry-After: 60`** は「今は無理、後で来い」の意味の標準ヘッダ。締め出しの返事ではない。

補足: 2026-07-20は **522**(本体が全く応答しない)、2026-07-21は **502**(本体が応答するが中身が壊れている)。
**症状が変化している**=向こう側で誰かが復旧を試みている気配はある。

---

## 3. ただし、ここは正直に言う(断定しない部分)

### 3.1 「復旧する」とは言えない
da.gd は**無料・個人運営規模の外部サービス**。1日以上の全断は、そのまま**畳んだ**可能性もある。
復旧時期を約束できる主体がどこにも居ない。**こちらから制御できない**のが本質的な問題。

### 3.2 これは INC-108 で既に「恒久策=完了」と書いた件の**残骸**
新規リンクは 5mgl.com / yoz2.com に切替済み=**normal path から da.gd は消えている**。
残っているのは**切替前に発行済みの53本**だけ。つまり新しい傷は増えていない。

### 3.3 ★BANの可能性そのものを消したわけではない
今回否定したのは「**da.gdがChamiを締め出した**」という筋だけ。
YouTube側のリンクポリシー・FANZA側の規約に関するリスクは**この調査とは無関係に別途存在する**
(`CLAUDE.md` §6.1 規約防衛P0)。Geminiの指摘を「全部ハズレ」として片付けない。

## 4. 実害と、やるべきこと

**実害= YouTube概要欄の53本のリンクが死んでいる=その53本の収益導線がゼロ。**
`local/dagd_swap_list.md` に旧(da.gd)→新(r2)の対応表が53行ある。
サンプル3本を実測し、**全て正常に302転送する**ことを確認済み。計測コードは同一なので
**クリック計測は途切れない**(中間ホップが1つ消えるだけ)。

→ **推奨: 復旧待ちをやめて差し替える。** 待つ理由(=すぐ直る見込み)がもう無い。
   作業はYouTube Studioでの概要欄置換=**Chamiの手作業**(このリポジトリにYT APIの口は無い)。

## 5. 未解明(Chamiへの確認事項)

- Cloudflare / ドメインレジストラ / GitHub Actions

★**da.gd経由ではない**ことだけは確認済み: GAS の `daGdShorten_()`(`gas/コード.gs:705`)は
`muteHttpExceptions` + `try/catch` で囲まれており、**da.gdが何を返しても例外を投げない**
(失敗時は空文字を返して長いURLのまま記録される)。よってGASの失敗メールの原因はこれではない。
