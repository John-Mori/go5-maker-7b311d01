# 設計書：投稿短縮URLの画面表示 ＋ YouTube説明欄

> 対象：5秒動画メーカー スマホ版（現行 v=22）。本機能で v=23 に上げる。

---

## 1. 目的

1. **動画作成後にBlueskyへ自動投稿した後**（および「今すぐ投稿」後）、その**投稿URLを短縮**し、**短縮URLをこのページ上にそのまま表示**（コピー可）する。
2. 「🦋 投稿」タブの**最下部に「YouTube説明欄」**を追加。**短縮URLをテキストとして自動で差し込み**、**そのままコピー**してYouTubeの概要欄に貼れるようにする。Blueskyの投稿本文欄と同じ感覚（編集可・永続・コピー可）。

---

## 2. 前提（既存実装の活用）

- 投稿成功時、`bluesky.js` が `bluesky-posted` イベントを発火 → 既存リスナが `recordToSheet()` でGASへ記録。
- **GAS（`gas/コード.gs`）の `doPost` は既に `{ ok:true, shortUrl, bitlyId }` を返している**（Bitlyで投稿URLを短縮済み）。
- すなわち**短縮URLはGASの応答として取得済み**。現在クライアントが応答を捨てているだけ＝それを使えばよい。**GAS本体・コア投稿ロジックは無変更**。

---

## 3. 機能A：短縮URLの画面表示

- 「🦋 投稿」タブ最下部に表示領域を追加：
  - `#shortUrlOut`（短縮URLを表示するコードボックス）＋「URLをコピー」ボタン。
  - 初期表示：「（投稿するとここに短縮URLが表示されます）」。
- 投稿後の流れ：`bluesky-posted` → `recordToSheet()` の応答 `shortUrl` を取得 → `#shortUrlOut` に表示。
- **動画作成タブの自動投稿ステータス**（`#bskyStatus`）にも短縮URLを併記（その場で見える・コピー可リンク）。

### GAS未設定時のフォールバック
- GAS未設定（短縮不可）の場合は、**投稿URL（フル）をそのまま表示**し、注記「短縮にはGAS設定が必要」を出す。機能は壊さない。

---

## 4. 機能B：YouTube説明欄（投稿タブ最下部）

- `#ytDesc`：編集可能な textarea（Blueskyの本文 `#bskyText` と同じ作り）。`localStorage["yt_desc"]` に保存・復元。
- 既定の文面（編集可）：
  ```
  おすすめ漫画を紹介しています📚
  ▼続き・詳細はこちら

  #漫画 #PR
  ```
- **投稿後、短縮URLを説明欄へ自動で差し込む**：
  - 直前に差し込んだ短縮URL（`prevShortUrl`）が説明欄内にあれば**新しい短縮URLへ置換**（常に最新1件に同期）。
  - 無ければ末尾に1行追加。
  - 反映後 `localStorage` に保存。
- 「短縮URLを挿入」ボタン（手動で末尾挿入）＋「説明欄をコピー」ボタン。
- コピーは `navigator.clipboard.writeText`（失敗時は一時textarea＋`execCommand` フォールバック。既存`.copy-btn`流用）。

> 「最初はBluesky投稿本文みたいなシステムでよい」に従い、説明欄は**固定文テンプレ＋短縮URL**の編集可テキストとし、そのままコピーできる形にする。将来、説明欄テンプレの拡充（章立て・複数リンク等）は容易に追加できる。

## 5. 影響範囲

| ファイル | 変更 |
|---|---|
| `index.html` | 投稿タブ最下部に「短縮URL表示＋YouTube説明欄」カードを追加。アセット `?v=22`→`?v=23`。 |
| `bluesky.js` | `bluesky-posted` リスナで `recordToSheet()` の応答 `shortUrl` を使用→表示＆説明欄差し込み。`#ytDesc` の保存/復元・コピー・挿入ボタン配線。`#bskyStatus` に短縮URL併記。 |
| `style.css` | `.short-row` / `.short-url` 等の小スタイル。`?v=23`。 |
| `gas/`・`bluesky-core.js`・`scheduler.js`・`integration.js`・`verify*` | **無変更**。 |

---

## 6. 受け入れ条件

- [ ] 動画作成→自動投稿後、短縮URLがページに表示され、コピーできる（GAS設定時）。
- [ ] 「今すぐ投稿」後も同様に短縮URLが表示される。
- [ ] 投稿タブ最下部にYouTube説明欄があり、短縮URLがテキストとして自動で入る。
- [ ] 説明欄はそのままコピーでき、編集・永続（再読込で保持）できる。
- [ ] GAS未設定でも壊れない（投稿URLをフォールバック表示＋注記）。
- [ ] 既存の投稿/スケジュール/検証/座標系は回帰なし。`?v=23` に更新。
