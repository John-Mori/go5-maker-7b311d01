# 投稿前チェック(規約防衛)

## なぜこれが最優先か
YouTube/FANZAの規約違反はチャンネル凍結・アカウント停止に直結する。チャンネルを失えば、再生数施策もコピー改善も全て無になる。だから「再生数を上げる」より「アカウントを守る」が先。このSkillは、投稿という不可逆な操作の前に置く最終ゲート。

## チェックリスト(全項目クリアで投稿可)

### 1. 露出基準 — 電車内で見られて困らないコマだけ
1コマ目・サムネ・全コマを点検。性的示唆(挑発ポーズ・下着相当の露出)が入っていないか。
なぜ: 性的示唆コマは年齢制限がかかり、Shortsフィード配信が実質ゼロになる(2025/8からAI年齢推定も稼働)。配信されなければ投稿の意味がない=除外と同義。

### 2. FANZAリンクの2段クッション — 直リンクを表に出さない
YouTube説明欄・固定コメントにFANZA直リンク(生アフィリンク)を貼っていないか。
正しい構造: YouTube → 短縮URL(r2/da.gd) → Bluesky投稿 → FANZA。生アフィリンクが出てよいのは Bluesky投稿本文のみ。
なぜ: YouTubeの外部リンクポリシー。アダルト外部サイトへの直接誘導は違反。

### 3. 煽り文言なし
題名・説明欄・コメントに「YouTubeでは見せられない続きは…」型の誘導がないか。
なぜ: 外部リンクポリシーで名指しの違反パターン。

### 4. 2チャンネル重複なし
acc1(月詠み)とacc2(宵桜艶帖)へ同一動画をコピー投稿していないか。作品・コマ・文言が変えてあるか。
なぜ: inauthentic/スパム判定(2025/7 収益化ポリシー)の火種。

### 5. 音源の焼き込みなし
生成動画に音源(BGM)が焼き込まれていないか(無音が正しい仕様)。
なぜ: BGMはYouTubeアプリのShortsエディタで「許可された音楽」を付けるのが正式運用。焼き込みは著作権リスク。

## 出力の型
各項目に ✅(問題なし)/⚠️(要修正)を付けて報告する。⚠️が1つでもあれば投稿を止め、何をどう直すかを具体的に示す。全て✅なら「投稿可」と明示。

例:
```
投稿前チェック(acc2・作品XXX)
1. 露出基準 ✅ 電車内OK
2. 2段クッション ✅ 説明欄は短縮URLのみ
3. 煽り文言 ⚠️ 説明欄「続きは…」を削除して
4. 2ch重複 ✅ acc1とはコマ・文言が別
5. 音源焼き込み ✅ 無音
→ 3を直せば投稿可
```
