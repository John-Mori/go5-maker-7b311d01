# 販促ラベル(セールラベル) テンプレート画像

2026-07-16 Chami提供の完成デザインへ差し替え(指示書「セールラベル数値可変化」)。
**数字なしの完成テンプレPNGを敷き、数字だけを `promo-label.js` がCanvasで描く**。

## ファイル(このフォルダ直下・全て透過PNG)
| ファイル | チャンネル | 種類 | 数字スロット(slot) |
|---|---|---|---|
| `tsukuyomi-discount-base.png` | 月詠み(acc1) | 割引率(今なら◯%OFF作品) | 「今なら」と「%」の間 |
| `tsukuyomi-price-base.png` | 月詠み(acc1) | 価格(月影に綴る¥◯作品) | ¥の右 |
| `yoizakura-discount-base.png` | 宵桜(acc2) | 割引率(今だと◯%OFF) | 「今だと」と「%OFF」の間 |
| `yoizakura-price-base.png` | 宵桜(acc2) | 価格(今宵の¥◯作品案内) | ¥と「作品案内」の間 |

- 原本はChamiがChatGPTで生成(市松柄が焼き込まれていたため、フラッドフィルで透過化済み)。
- slot座標は基材の画素解析で確定し `promo-label.js` の `TEMPLATES` に記載。**基材を差し替えたらslotも再測定**すること。
- 数字の質感(クリーム金/桜白のグラデ+縁+光彩)はコード側(`drawDigits`)。
  Chami提供の数字シート(0〜9)は背景が焼き込みで切り出すと継ぎ目が出るため不採用。
  `local/promo-ref/` に保管(将来、完全透過の数字素材が来たらスプライト方式へ精密化できる)。

## 動作(promo-label.js)
- 種類はUIのセレクタで切替: ◯%OFF(割引率) / ¥価格。localStorage `promo_label_type`。
- セール中のみ表示。**定価の作品はチェックONでも非表示**。不正値(0/負/NaN)はラベルごと非表示。
- 1〜3桁は幅に合わせ自動縮小(スロット中央揃え・指示書§6)。
- 表示ON/OFF=チェックボックス(既定ON・新規作成リセット後もONへ復帰)。
- 位置=D-pad+指ドラッグ / 大きさ=±ボタン。フレーム比で保存(プレビュー=書き出し一致)。
